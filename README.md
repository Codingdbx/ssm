## 简单的SSM框架（实现增删改查及分页）项目


### 实现功能：
- 1.增删改查
- 2.实现了分页功能
- 3.带条件分页 (即点击下一页条件不丢失)

### 运用知识

- 数据库 mysql
- 数据库连接池 c3p0
- 运用jackson.jar包将对象转为json格式
- 标签库 jstl标签库 和 el表达式
- mcv三层框架的分离

## 开发环境
- myeclipse-2014 
- spring 3.0
- springmvc 2.0
- mybatis 3.0
